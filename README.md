# HttpCache extension for Spring MVC

The HttpCache extension for Spring MVC is a library which aim to extends Spring MVC in order to add `cache` headers to HTTP requests of controllers endpoints annotated with `@HttpCache`. The library contains the also [the definition of the `@HttpCache` annotation](link/to/HttpCache/source)

## How to use it

- Add to dependencies
- Anotate Controller methods
